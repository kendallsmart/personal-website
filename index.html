My Projects

Brick Breaker

This project is a brick breaker game coded in C. The user plays the game by launching the ball by hitting the space bar on their computer, and moves the bottom block around via the cursor in order to try and hit all of the blocks with the ball.

Pinball

This project is a Pinball game coded in C++. The user plays the game by launching the ball by hitting the space bar and the ball's speed and direction are affected depending on which of the objects it encounters as it travels down the middle of the screen.
